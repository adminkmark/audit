# Аналізатор форматування курсових робіт (PDF)

Цей додаток написано на Python з використанням бібліотеки **Streamlit** для веб-інтерфейсу та **PyMuPDF** для аналізу PDF-файлів.

Він автоматично перевіряє завантажену курсову роботу на дотримання наступних вимог згідно з методичкою:
- Поля сторінки (Ліве 2.5 см, Праве 1.0 см, Верхнє 2.0 см, Нижня 2.0 см).
- Основний шрифт — Times New Roman.
- Розмір шрифту — 14.

## Як запустити додаток локально?

1. Переконайтесь, що встановлено Python 3.
2. Відкрийте термінал (або командний рядок) у цій папці.
3. Якщо ви ще не встановлювали залежності, виконайте:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустіть додаток командою:
   ```bash
   streamlit run app.py
   ```
5. У вашому браузері автоматично відкриється сторінка додатку (зазвичай `http://localhost:8501`).

## Як завантажити це в Інтернет (для студентів)?

Цей додаток можна абсолютно безкоштовно розмістити в хмарі, щоб студенти мали доступ до нього звідки завгодно:
1. Завантажте всі ці файли (`app.py`, `requirements.txt`) у новий репозиторій на GitHub.
2. Авторизуйтесь на сайті [share.streamlit.io](https://share.streamlit.io/).
3. Натисніть *"New app"*, виберіть ваш репозиторій, і Streamlit безкоштовно запустить ваш додаток в мережі Інтернет!
